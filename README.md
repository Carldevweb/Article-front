# Article Front (Angular)

Frontend du projet **Article** : affichage des articles/catégories, authentification JWT, pages admin, upload d’images, UI inspirée Figma.

## Stack
- Angular 19
- TypeScript
- RxJS
- Bootstrap (et styles UI Figma)

## Prérequis
- Node.js + npm
- Backend lancé en local (Spring Boot)

## Installation
```bash
npm install
